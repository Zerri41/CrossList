Ligar ao Google Drive — 3 passos
Faz isto uma vez e a sincronizacao fica automatica para sempre.
1
Criar Service Account no Google
Vai a console.cloud.google.com → Novo projecto → Activar Drive API → IAM → Service Accounts → Criar → Chave JSON → Download
2
Partilhar a pasta com o email da Service Account
No Google Drive, clica direito na pasta Inventario → Partilhar → cola o email da service account (ex: crosslist@projecto.iam.gserviceaccount.com)
3
Colar as credenciais aqui
Abre a pasta no browser, o ID esta no URL: drive.google.com/drive/folders/ESTE-E-O-ID
